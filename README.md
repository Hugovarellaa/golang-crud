### 📝 Clone and use

```bash
   # Clone this repositorie
   $ git clone https://github.com/Hugovarellaa/golang-crud

   # Enter in folder
   $ cd golang-crud

   # Run postgresDB
   $ docker-compose up

   # Run Go Application
   $ go run main.go
```
